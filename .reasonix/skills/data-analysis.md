# Data Analysis Skill

> 分析 Excel/CSV 数据文件，生成统计摘要、数据清洗、查询。
> 来源：Trae skills / `data-analysis`

## 核心能力

- 检查文件结构（表单、列、类型、行数）
- 执行 SQL 查询探索数据
- 生成统计摘要（均值、中位数、标准差、分位数、空值）
- 支持多表单 Excel
- 导出结果为 CSV/JSON/Markdown
- 处理大文件

## 工作流

### Step 1: 了解需求
- 文件路径和格式
- 分析目标（汇总、过滤、聚合、对比等）
- 输出格式要求

### Step 2: 检查文件结构
- 读取文件，查看表单/列/类型
- 查看前几行样本数据

### Step 3: 执行分析
- 根据需求构建分析查询
- 数据清洗和过滤
- 统计分析

### Step 4: 输出结果
- 格式化输出为 Markdown 表格
- 可导出 CSV/JSON

## 工具
- Python (pandas, openpyxl) 用于数据分析
- 可直接输出 Markdown 供文档使用
