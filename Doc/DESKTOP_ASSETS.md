# Fashion Store — 桌面端素材清单

> 本项目为数码/3C 电商平台，桌面端网页设计，风格：精品时尚电商（Clean Blue）。
> 面向用户：消费者购物流程，商品展示 + 下单。
> 设计关键词：简洁、专业、精致、轻奢感。

## 三、首页素材

### 3.1 轮播 Banner（Carousel）

| 尺寸 | 用途 |
|---|---|
| 宽度自适应，高度 **300px**，JPG/PNG | 首页顶部大 Banner，建议 3~5 张轮播 |
| 比例建议 16:9 或 3:1 | 确保在各类屏幕上清晰 |

**Banner 风格要求：**
- 时尚服装/模特为主视觉
- 色调统一，饱和度适中
- 文字 overlay：简洁促销语（如"夏季新品"、"限时特惠"）
- 建议尺寸：1920×600px 原图

### 3.2 商品卡片缩略图

| 尺寸 | 说明 |
|---|---|
| 正方形 1:1，宽度 >= 300px | 商品列表封面 |
| 文件格式：JPG/WebP，压缩率 80% | 背景白色或浅灰 |

### 3.3 分类图标

| 尺寸 | 用途 |
|---|---|
| 48×48 PNG，透明背景 | 首页侧边分类列表图标 |

## 十二、交付格式

请提供以下格式：

```
/assets
  /logos
    logo-icon.png (128x128)
    logo-full.png (高度40px)
  /banners
    banner-1.jpg (1920x600)
    banner-2.jpg
    ...
  /products
    /{goods_id}
      cover.jpg (800x800)
      gallery-1.jpg
      ...
  /placeholders
    product-placeholder.jpg (300x300, #F5F7FA背景)
  /icons
    (如需非emoji图标)
  /brand
    login-bg.jpg (1920x1080, 深色)
    avatar-default.png (60x60, 圆形)
```

---

## 十三、注意事项

1. 所有图片请压缩后提供，商品图单张不超过 300KB
2. 图片命名统一用英文或数字，禁止中文命名
3. Banner 图片避免文字遮挡核心内容（文字 overlay 单独提供 PSD 层）
4. 模特图优先选用亚洲面孔，场景真实自然
5. 如无法提供真实素材，请优先提供**纯色背景 + 商品轮廓的示例图**，我方可用 CSS 占位先跑通流程
