## 一、命名规范 / Naming Convention

### 轮播图 Banner
```
banner-{序号}-{中文主题关键词}.jpg
```
示例：`banner-1-夏季新款.jpg`、`banner-2-轻奢女装.jpg`

### 商品封面图 Product Cover
```
goods-{商品ID}-{商品名拼音}.jpg
```
示例：`goods-1-molan-di-mian-t.jpg`、`goods-3-zhi-xiu-niu-zai-ku.jpg`

### 占位图 Placeholder（备用）
```
placeholder-{类型}.jpg
```
